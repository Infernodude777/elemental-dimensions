### ‚≠ê BEST OPTIONS (Recommended)

#### 1. **Stable Diffusion Web UI** (100% Free, Unlimited)
- **URL**: https://github.com/AUTOMATIC1111/stable-diffusion-webui
- **Setup**: Runs locally on your PC
- **Pros**:
  - Completely free and unlimited
  - Full control over generation
  - Can batch generate all textures at once
  - No censorship or restrictions
- **Cons**:
  - Requires ~10GB disk space + GPU (works on CPU but slower)
  - Initial setup required
- **Best For**: Generating ALL textures in bulk with custom model

#### 2. **Bing Image Creator / DALL-E 3** (Free, Limited)
- **URL**: https://www.bing.com/images/create
- **Pros**:
  - Very high quality
  - Microsoft account only (free)
  - Good at understanding pixel art
- **Cons**:
  - Daily limit (~15-25 images/day)
  - May refuse some prompts
  - Need to generate textures in batches over several days
- **Best For**: High-quality individual textures

#### 3. **Leonardo.ai** (Free Tier)
- **URL**: https://leonardo.ai/
- **Pros**:
  - 150 free tokens daily
  - Pixel art model available
  - Batch generation support
- **Cons**:
  - Requires account creation
  - Limited daily tokens
- **Best For**: Quick batch generation with retries

#### 4. **Craiyon (Formerly DALL-E mini)** (100% Free)
- **URL**: https://www.craiyon.com/
- **Pros**:
  - Completely free, unlimited
  - No account required
- **Cons**:
  - Lower quality
  - Slower generation
  - May not be good for pixel art
- **Best For**: Quick tests, low priority textures

## üîß POST-GENERATION PROCESSING

After generating textures:

1. **Resize** all images to exactly 16x16 pixels (use any image editor)
2. **Remove backgrounds** if not transparent (use remove.bg or GIMP)
3. **Rename files** to match the guide (e.g., `inferno_stone.png`)
4. **Place in folders**:
   - Blocks ‚Üí `ResourcePack_ElementalDimensions/assets/elementaldimensions/textures/block/`
   - Items ‚Üí `ResourcePack_ElementalDimensions/assets/elementaldimensions/textures/item/`

## üì¶ AUTOMATION TOOLS

**Bulk Processing**:
- **ImageMagick** (command-line): Convert/resize all at once
- **XnConvert** (GUI): Batch resize, rename, format convert
- **GIMP** (Script-Fu): Automate transparency and resize

**Example ImageMagick Command** (resize all to 16x16):
```bash
magick mogrify -resize 16x16! -format png *.png
```
