2. **Paste into your AI tool** (Stable Diffusion, Bing Creator, Leonardo.ai, etc.)
3. **Generate** - You'll get a horizontal sprite sheet
4. **Download and slice** into individual files
5. **Rename files** according to TEXTURE_GUIDE.md
6. **Repeat** for all 15 prompts

### Recommended Order:
1. Start with **BLOCKS 1/5** (Dimension Stones) - Most important
2. Then **BLOCKS 2/5** (Dimension Ores) - Second most important
3. Then **ITEMS 1/5** (Essences & Hearts) - Core items
4. Continue with remaining blocks and items
5. Finish with mob sprites (least urgent)

### Post-Processing:

**For Horizontal Sprite Sheets:**
```bash
# ImageMagick - auto-slice horizontal sprite sheet
magick sprite.png -crop 16x1@ +repage texture_%d.png
```

**Or use online tools:**
- https://ezgif.com/sprite-cutter (Upload, set grid, download)
- https://www.imgonline.com.ua/eng/cut-photo-into-pieces.php

### Expected Results:
- Each prompt generates **8-18 textures** side-by-side
- Much shorter prompts = less likely to cut off
- Easier for AI to process
- Faster generation time
