# Elemental Dimensions - Quick Start Guide

## üöÄ Getting Started

### Installation
1. Make sure you have **Minecraft 1.21.1** with **Fabric Loader**
2. Install **Fabric API** and **GeckoLib** as dependencies
3. Copy `elemental-dimensions-1.0.0.jar` to your mods folder
4. Launch Minecraft!

### First Steps
Once in game, you have access to 13 different dimensions:

#### Original 6 Dimensions
1. **Firelands** - Volcanic realm of fire
2. **Aquatica** - Underwater ocean realm
3. **Terra Depths** - Underground earth realm
4. **Skyreach Peaks** - Sky islands realm
5. **Celestial** - Space-themed realm
6. **Void Realm** - Dark void dimension

#### Undead Realm
7. **Undead Realm** - Spooky graveyard dimension
   - 27 unique blocks
   - 15 special items
   - 11 custom mobs (including Lich King boss)
   - 15 structures

#### New Dimensions (6)
8. **Dreaming Depths** - Mystical dream realm
9. **Celestine Expanse** - Floating sky islands
10. **Fungal Dominion** - Overgrown mushroom forest
11. **Forgotten Archive** - Ancient library
12. **Astral Frontier** - Cosmic space realm
13. **Gloomy Caverns** - Dark underground caves

## üéÆ Essential Commands

### Teleportation
```
/edtp <dimension_name>
```
Examples:
- `/edtp firelands` - Go to fire dimension
- `/edtp dreaming_depths` - Go to dream realm
- `/edtp undead_realm` - Visit the undead

### Getting Items
```
/getallitems
```
Spawns a chest (or two!) with ALL mod items in front of you. Perfect for creative mode testing!

### Get Guidebook
```
/guide
```
Gives you a book with information about all 13 dimensions and their bosses.

### Finding Things
```
/edlocate structure <structure_name>
/edlocate biome <biome_name>
```
Examples:
- `/edlocate structure undead_crypt`
- `/edlocate biome dreaming_depths_plains`

### Spawning Mobs
```
/edspawn <mob_name>
```
Examples:
- `/edspawn lich_king` - Spawn the boss!
- `/edspawn zombie_miner`
- `/edspawn ghost`

### Help
```
/edhelp
```
Shows all available commands.

## üêõ Troubleshooting

### Mod won't load
- Check you have Fabric API and GeckoLib installed
- Verify Minecraft version is 1.21.1
- Check logs for errors

### Commands not working
- Make sure you have operator permissions
- Most commands require level 2 permissions
- Use Tab completion to verify command syntax

### Textures look wrong
- Install the resource pack for best visuals
- Reload resource packs (F3+T)
- Check resource pack is enabled and at top of list

### Can't find structures
- Structures only generate in their respective dimensions
- Use `/edlocate structure <name>` for hints
- Some structures are rare - explore more!
