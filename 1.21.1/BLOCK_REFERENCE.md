## Quick Stats
- **Total Blocks**: 129 unique blocks
- **Dimensions**: 7 (Undead Realm + 6 new)
- **Average per Dimension**: ~18 blocks
- **All blocks have**:
  - Blockstate JSON
  - Block model
  - Item model
  - Loot table
  - 16x16 texture
  - English translation

## How to Get Blocks
Use the `/getallitems` command to spawn chests containing all blocks and items!

## Texture Information
- All textures are 16x16 PNG format
- Color-coded by dimension:
  - Dreaming Depths: Lavender/Purple (#D8BFD8, #E6E6FA)
  - Celestine Expanse: Cyan/Blue (#87CEEB, #B0E0E6)
  - Fungal Dominion: Green/Brown (#556B2F, #8FBC8F)
  - Forgotten Archive: Gold/Amber (#DAA520, #F0E68C)
  - Astral Frontier: Purple/Magenta (#9370DB, #BA55D3)
  - Gloomy Caverns: Gray/Dark (#696969, #2F4F4F)
- Available in resource pack: `resourcepack/assets/elementaldimensions/textures/block/`
