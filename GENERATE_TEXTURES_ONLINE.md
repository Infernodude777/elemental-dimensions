## üîÑ **After Generation - Resize to 16x16**

The generated images will be 512x512. You need to resize them:

### **Option 1: Online Tool (Easiest)**
1. Go to: https://onlinepngtools.com/resize-png
2. Upload the generated image
3. Set width: 16, height: 16
4. Set resize method: **Nearest Neighbor** (important for pixel art!)
5. Download the 16x16 version

### **Option 2: Bulk Resize Tool**
1. Go to: https://bulkresizephotos.com/
2. Upload all 11 images
3. Set dimensions: 16 x 16 pixels
4. Set algorithm: Nearest Neighbor
5. Download all as ZIP
