# Elemental Dimensions - Texture Creation Guide

**IMPORTANT NOTE**: The AI assistant cannot create PNG image files. This guide provides specifications for what textures you need to create manually using image editing software like GIMP, Photoshop, or dedicated Minecraft texture editors like BlockBench.

## Required Texture Dimensions
- **Items**: 16x16 pixels
- **Blocks**: 16x16 pixels (6 faces per block if different)
- **GUI Elements**: Varies (typically multiples of 16)

All textures should be PNG format with transparency support.

---

## 1. INFERNO REALM (Fire Dimension)

### Theme: Scorching volcanic wasteland with molten lava and burning stone

### Blocks Needed:
- **Inferno Stone** (`inferno_stone.png`)
  - Base color: Dark charcoal gray (#2B2B2B)
  - Accent: Glowing orange-red cracks (#FF4500, #B22222)
  - Style: Rocky texture with lava veins
  - Reference: Basalt + glowing lava cracks

- **Fire Ore** (`fire_ore.png`)
  - Base: Inferno stone texture
  - Ore spots: Bright orange-yellow crystals (#FFA500, #FFD700)
  - Effect: Add subtle glow/emission
  - Style: Embedded flame-like crystals

### Items Needed:
- **Fire Essence** (`fire_essence.png`)
  - Shape: Glowing ember/flame droplet
  - Colors: Orange to red gradient (#FF4500 → #8B0000)
  - Effect: Add particle sparkles around edges

## TEXTURE CREATION TIPS

1. **Color Palette**: Stay consistent with dimension theme
2. **16x16 Grid**: All textures must be 16x16 pixels
3. **Transparency**: Use alpha channel for see-through elements
4. **Shading**: Add depth with 2-3 shade variations
5. **Edges**: 1-pixel borders for definition
6. **Style**: Keep Minecraft's blocky aesthetic
7. **Testing**: Load in-game to verify appearance
